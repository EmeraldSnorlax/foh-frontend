> We use Yarn, not npm!

Make sure to set the location of your foh server [here](https://github.com/EmeraldSnorlax/foh-frontend/blob/b508ed67239a15e5668b92f91fcb356816adfa7e/src/index.ts#L5).

Development
---
```
yarn install
yarn start
```

Building for prod:
---
```
yarn install
yarn build
```

Resulting files are in `./build`.
